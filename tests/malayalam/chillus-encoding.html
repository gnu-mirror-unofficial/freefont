Three representations of chillus in Malayalam

Two encodings have been recognized as the Unicode standard for the combination 'ṉṯa' which appears as a letter chillu-n with a subscript letter rra:

ന്റ ← legacy, U+0d28 U+0d4d U+0d31

The modern encoding, adopted in Unicode 5.1 uses a special chillu character:

ൻ്റ ← modern, U+0d7b U+0d4d U+0d31

(Note: the main Malayalam font in MS Windows, Kartika, does not display the modern form properly, although as of 2020 it is dated 2015.)

Another encoding

ന്‍റ ← U+0d28 U+0d4d U+200d U+0d31

had often been used to represent the same conjunct, but it conflicts with an encoding of a different conjunct, 'nta', where two letters appear side-by-side; side-by-side form was the legacy Unicode interpretation of the encoding. As of Unicode 5.1 the preferred encoding of the side-by-side combination is:

ൻറ ← modern, U+0d7b U+0d31

There were legacy encodings for five simple chillus, which were replaced by atomic code points:

ണ്‍ ← legacy, U+0d23 U+0d4d U+200d
ൺ ← modern, U+0d7a
ന്‍ ← legacy, U+0d28 U+0d4d U+200d
ൻ ← modern, U+0d7b
ര്‍ ← legacy, U+0d30 U+0d4d U+200d
ർ ← modern, U+0d7c
ല്‍ ← legacy, U+0d32 U+0d4d U+200d
ൽ ← modern, U+0d7d
ള്‍ ← legacy, U+0d33 U+0d4d U+200d
ൾ ← modern, U+0d7e

There was no legacy Unicode encoding for the ka-chillu (although many fonts support a non-standard encoding similar to those for the other chillus.)

ക്‍ ← non-standard U+0d15 U+0d4d U+200d

The modern encoding is the atomic code point:

ൿ ← modern, U+0d7f

Unicode 9.0 added three archaic chillus as atomic code points. (Few fonts support these at the time of this writing.)

ൔ ← rare, U+0d54
ൕ ← rare, U+0d55
ൖ ← rare, U+0d56
