joiner tests in Malayalam

ZWNJ after halant: explicit candrakala, inhibit ligatures.

ZWNJ before halant: request open form conjunct (coherant only when next consonant is ra.)

ZWJ before halant: request traditional ligatures.

consonant with ra, joiners request open and traditional forms

Using a font fully supporting the joiners, the three last columns should be different. The second column will differ according to orthography.
