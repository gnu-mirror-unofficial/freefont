stacked conjuncts with ra-tail in Malayalam

This is meant to list the consonant conjuncts with -ra which actually appear in the language. I started with glyphs from existing fonts supporting traditional orthography, taking those whose subscript consonant sported a ra-tail.

Note there is no ra form for any of
nga ca nja ttha nna nnna ya ra rra la lla llla ttta.

ZWNJ conjuncts

A ZWNJ meant to request "atomic" display of conjuncts shouldn't break reformed pre-base ra in Reformed orthography. Here, candrakala should appear between consonants, and the prefix ra before the final consonant. Vowels should bracket the final consonant.
