ZWJ forms with ra in Malayalam

This is meant to list all the consonant and conjunct forms in the font to which a tradtional ra tail is applied, and the effect that an intervening ZWJ character has on them.

Note that not all consonants, and not all consonant conjuncts, normally occur with -ra in Malayalam. I don't think there is any rule that says that one must not ever put a ra-tail on a given letter such as nya -- the combination just never comes up.

Like these forms:

k.ra ക്ര traditional, no joiner

k.ra ക‍്ര traditional, with ZWJ

k.ra ക‍്ര reformed, with ZWJ

but in contrast to these

k.ra ക‌്ര traditional, with ZWNJ

k.ra ക്ര reformed, no joiner
