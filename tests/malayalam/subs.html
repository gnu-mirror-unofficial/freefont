subscripted forms by request

Many consonant clusters are not found in proper Malayalam words, but a need to display them as a conjunct may occasionally arise (e.g. in a person's name.)

For a select subset of consonants, a subscripted form is available using the ZWJ joiner. By default, it is better to display these in "atomic" form with visible candrakala.

nn.pa ണ്പണ്പോ ണ‍്പണ‌്പ
