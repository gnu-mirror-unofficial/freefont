ZWNJ tests in Malayalam

ZWNJ after halant: explicit candrakala, inhibit ligatures.

ZWNJ before halant: request open form conjunct (coherant only when next consonant is ra.)

ZWJ before halant: request traditional ligatures.

ZWNJ consonant with va, request of post-base, reformed form

There is a single case of a special conjunct form with -va. This is typically rendered with a post-base form in reformed orthography.

plain req. rfmd. atomic
lll.va ഴ്വ ഴ‌്വ ഴ്‌വ

ZWNJ consonant with ya, request of post-base form

The one case of a ligature with ya is y.ya which is one of the ligatures prescribed in the reform order. (I don't know of any specific use for an "open" form. But in principle:)

plain req. open atomic
y.ya യ്യ യ‌്യ യ്‌യ
